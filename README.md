Developed a Retrieval-Augmented Generation (RAG) chatbot that enables conversational question-answering over PDF documents using Streamlit, LangChain, FAISS, and Hugging Face embeddings.

Built an end-to-end document processing pipeline for PDF text extraction, chunking, embedding generation and vector database creation, enabling accurate context-aware information retrieval.

Integrated the Gemini API to generate intelligent, context-grounded responses based on retrieved document content.

Designed the application to support both cloud-based LLMs and locally hosted Llama models, providing flexibility for privacy focused and personalized AI deployments.

Deployed the application on Streamlit Cloud delivering an interactive and scalable web based user experience.
